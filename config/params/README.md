This folder stores the parameter setup for MadGraph process, especially the branching ratios for BSM particle decays and BSM couplings.
- [RPV_C1Hl_N1WlZvHv](https://atlas.web.cern.ch/Atlas/GROUPS/PHYSICS/PAPERS/SUSY-2018-36/): 1000022 (N1 inclusive decay to W/Z/H) and 1000024 (C1 -> H + lepton).
- [RPV_C1Hl](https://atlas.web.cern.ch/Atlas/GROUPS/PHYSICS/PAPERS/SUSY-2018-36/): 1000024 (C1 -> H + lepton) * 2.
- [thFCNC_ctH_tcphi, thFCNC_utH_tphi](https://atlas.web.cern.ch/Atlas/GROUPS/PHYSICS/PAPERS/HIGG-2016-26/): Hqt FCNC in tH production.
- [ttFCNC_tcH_tcphi, ttFCNC_tuH_tphi](https://cms-results.web.cern.ch/cms-results/public-results/publications/TOP-19-002/index.html): Hqt FCNC in tt production.
