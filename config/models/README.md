This folder stores the UFO models for MC generation.
- [RPVMSSM_UFO_AM](https://github.com/elliot-hughes/rpvmssm_ufo) from Angelo Monteux. Renamed from `RPVMSSM_UFO` to avoid overriding the original model in the official repo. Includes RPV vertices for SUSY MSSM.
- [TopFCNC](https://cp3.irmp.ucl.ac.be/projects/feynrules/wiki/TopFCNC). Official model for the FCNC top process.
- [ZpHA](mailto:mawatari@iwate-u.ac.jp) from K. Mawatari. Massive neutral boson decays to the Higgs and photon.
